¿Cómo funciona?

  1. 1. Obtén un archivo .txt con todas tus preguntas. Quizás necesites transformar tu archivo PDF o Word a .txt, para ello, puedes usar esta herramienta para PDF a .txt o esta herramienta para Word a .txt.

  2. 2. Sube tu archivo .txt con todas las preguntas del test y rellena los diferentes parámetros. Es importante que los rellenes bien para la lectura correcta del documento.

  3. 3. Al hacer clic en "¡Adelante!" un script realizará una selección de las preguntas deseadas de forma aleatoria y se te descargará automáticamente el archivo .txt con dichas preguntas.

  4. 4. Ahora necesitas transformarlo a PDF para poder ser importado en Microsoft Forms. Puedes hacerlo a través de esta herramienta.

  5. 5. Ahora importa el PDF en Microsoft Forms haciendo clic en la opción "Importación rápida" o "Quick import", donde tendrás que subir tu PDF y seleccionar la opción "Cuestionario" o "Quiz".
